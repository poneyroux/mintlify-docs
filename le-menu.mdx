---
title: "Le menu"
---

Le menu de navigation est un élément structurant du site : il permet aux visiteurs de se repérer et d'accéder aux différentes sections. Dans WordPress, les menus sont entièrement personnalisables : on choisit librement les éléments qui les composent, leur ordre, leur hiérarchie, et l'endroit du site où ils s'affichent. 

La gestion des menus s'effectue depuis Apparence \> Menus dans le back-office.

## Création d'un menu

### Créer un nouveau menu

Lors de la première visite sur l'écran Menus, si aucun menu n'existe encore, WordPress propose directement d'en créer un. Si des menus existent déjà, un sélecteur en haut de page permet de basculer de l'un à l'autre.

1. Cliquer sur le lien Créer un nouveau menu.
2. Saisir un nom de menu dans le champ prévu (ce nom est uniquement visible dans le back-office, pas sur le site).
3. Cliquer sur Créer le menu.

### Ajouter des éléments au menu

Une fois le menu créé, le panneau de gauche propose plusieurs types d'éléments à y ajouter :

- Pages : les pages statiques du site (Accueil, Contact, Mentions légales, etc.).
- Articles : un article spécifique peut être directement intégré dans le menu.
- Catégories : pour renvoyer vers la liste des articles d'une catégorie donnée.
- Liens personnalisés : pour insérer une URL externe ou une ancre, avec un texte d'affichage libre.

L’ancre (anchor) est une balise que l’on place au sein d’une page. Elle permet de placer un lien directement vers cette ancre, ce qui peut être pratique en cas de site avec une seule page.

1. Dans le panneau de gauche, sélectionner le type d'élément souhaité (Pages, Catégories, Liens personnalisés, etc.).
2. Cocher les éléments à ajouter, ou renseigner l'URL et le texte pour un lien personnalisé.
3. Cliquer sur Ajouter au menu.
4. Les éléments apparaissent dans la zone de construction du menu, à droite.

### Organiser les éléments du menu

Les éléments du menu peuvent être réorganisés par glisser-déposer dans la zone de construction. Vous pouvez alors choisir d’avoir des menus parents ou des menus enfants (sous-menu)

Exemple :

    Accueil

    Nos produits

        ├── Catégorie A        ← sous-menu

        └── Catégorie B        ← sous-menu

    Blog

    Contact

    

Chaque élément du menu peut être renommé en cliquant sur la flèche à droite de son bloc, ce qui ouvre ses options : Texte de navigation (le label affiché sur le site), Titre, et bouton Supprimer.

### Créer un menu hors champ

Il est possible de créer un menu sans lui assigner d'emplacement sur le site. Ce type de menu, dit « hors champ », n'est pas affiché automatiquement sur le front-office mais peut être inséré manuellement dans un widget ou un bloc de l'éditeur. Cette pratique est utile pour créer, par exemple, un menu de pied de page secondaire ou un menu de liens rapides dans la colonne latérale.

### Comprendre les emplacements

<Warning>
  Un menu créé mais non assigné à un emplacement n'apparaît pas sur le site. L'assignation est une étape obligatoire.
</Warning>

###  Assigner un menu à un emplacement

L'assignation peut se faire de deux façons :

<u>Méthode 1 — Depuis l'écran Menus :</u>

1. Aller dans Apparence \> Menus.
2. Cliquer sur l'onglet Gérer les emplacements en haut de page.
3. Pour chaque emplacement listé, sélectionner le menu correspondant dans le menu déroulant.
4. Cliquer sur Enregistrer les modifications.

<Frame>
  ![IV 2 EMPLACEMENTSVIAMENUS](/images/IV.2.EMPLACEMENTSVIAMENUS.png)
</Frame>

<u>Méthode 2 — Depuis le Personnaliseur :</u>

1. Aller dans Apparence \> Personnaliser \> Menus.
2. Cliquer sur le menu à configurer.
3. Cocher l'emplacement souhaité dans la section Emplacement du menu.
4. Cliquer sur Publier.
