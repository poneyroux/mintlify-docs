L'apparence d'un site WordPress est définie par son thème. Un thème est un ensemble de fichiers qui détermine la mise en page, les typographies et les options visuelles du site. Dans le cadre de l'épreuve E5B, le thème utilisé est Astra.

Astra est l'un des thèmes les plus populaires de l'écosystème WordPress. Il est reconnu pour sa légèreté, sa compatibilité avec l'éditeur Gutenberg et ses nombreuses options de personnalisation accessibles sans compétences techniques. La plupart des réglages visuels sont accessibles depuis le Personnaliseur, l'interface de configuration du thème.

Pour accéder au Personnaliseur : Apparence \> Personnaliser dans le menu du back-office.

## Personnaliser l'apparence

Le Personnaliseur (ou Customizer) est l'interface centrale de réglage de l'apparence du site. Il se présente sous la forme d'un panneau latéral à gauche, avec un aperçu en temps réel du site à droite. Toute modification est visible instantanément avant d'être enregistrée.

Avec le thème Astra, le Personnaliseur est organisé en plusieurs sections : identité du site, en-tête, couleurs, typographie, mise en page, colonne latérale, pied de page, réseaux sociaux, etc.

<Warning>
  Les modifications effectuées dans le Personnaliseur ne sont appliquées sur le site qu'après avoir cliqué sur le bouton Publier en haut du panneau.
</Warning>

## Identité du site

L'identité du site regroupe les éléments d'identification visuelle fondamentaux, accessibles via Personnaliser \> Identité du site.

- Titre du site : le nom du site, affiché dans l'onglet du navigateur et repris par les moteurs de recherche.
- Slogan : une courte phrase décrivant l'activité ou le positionnement du site. Il peut être masqué si nécessaire.
- Logo : l'image représentant la marque. Astra permet de régler la largeur d'affichage du logo directement depuis ce panneau.
- Icône du site (favicon) : la petite image carrée affichée dans l'onglet du navigateur et dans les favoris.

## Modifier l'en-tête

L'en-tête (header) est la zone supérieure du site, généralement visible sur toutes les pages. Avec Astra, ses options sont accessibles via Personnaliser \> En-tête.

Il est possible de modifier :

- Le texte de l'en-tête (titre, slogan).
- L'image d'arrière-plan : une photo peut être appliquée en fond de l'en-tête pour renforcer l'identité visuelle.
- La vidéo d'arrière-plan : Astra permet également d'utiliser une vidéo en fond d'en-tête pour un rendu plus dynamique.
- La hauteur, l'alignement du contenu et la superposition (overlay) d'une couleur sur l'image ou la vidéo sont également paramétrables selon la version d'Astra utilisée.

## Couleurs

La section Couleurs du Personnaliseur permet de définir la palette visuelle globale du site. Avec Astra, les principaux réglages disponibles sont :

- Couleur d'accentuation : couleur principale appliquée aux boutons, liens et éléments interactifs.
- Couleur de fond : couleur d'arrière-plan des pages.
- Couleur du texte : couleur par défaut du contenu textuel.

Il est conseillé de choisir des couleurs cohérentes avec la charte graphique de l'entreprise représentée dans le sujet d’examen.

<Warning>
  Toutes les couleurs se règlent à cet endroit. Vous ne devez pas essayer de changer les couleurs d’un titre spécifique sur une page : il s’agit d’une mauvaise pratique.
</Warning>

### Réglage de la page d'accueil

Par défaut, WordPress affiche les derniers articles publiés sur la page d'accueil. Il est possible de modifier ce comportement et d'afficher à la place une page statique créée au préalable.

Ce réglage est accessible depuis deux endroits :

- Réglages \> Lecture dans le back-office.
- Personnaliser \> Page d'accueil dans le Personnaliseur.

| Option                    | Description                                                                                   |
| ------------------------- | --------------------------------------------------------------------------------------------- |
| **Vos derniers articles** | L'accueil affiche les articles publiés dans l'ordre chronologique inverse (option par défaut) |
| Une page statique         | L'accueil affiche une page choisie parmi les pages existantes du site.                        |

<Frame>
  ![III 5 RÉGLAGES PAGE ACCUEIL](/images/III.5.RÉGLAGESPAGEACCUEIL.png)
</Frame>

Lorsque l'option « Page statique » est sélectionnée, il faut désigner deux pages :

- La page d'accueil : la page affichée sur la racine du site.
- La page des articles : la page où sera affiché le flux des articles (optionnel).

## Colonne latérale

La colonne latérale (sidebar) est une zone d'affichage secondaire placée à côté du contenu principal, généralement sur les pages d'articles. Elle est composée de widgets : des blocs modulaires que l'on peut ajouter, supprimer et réorganiser.

### **Configurer les widgets**

Les widgets de la colonne latérale sont gérables depuis Apparence \> Widgets dans le back-office. Les widgets disponibles par défaut incluent : recherche, articles récents, catégories, archives, texte, etc.

### **Choisir la position de la colonne latérale**

Avec Astra, la position et l'affichage de la colonne latérale sont paramétrables via Personnaliser \> Mise en page \> Blog / Articles individuels :

- Colonne latérale à droite (par défaut)
- Colonne latérale à gauche
- Pas de colonne latérale (mise en page pleine largeur)

Ce réglage peut s'appliquer globalement à tout le site, ou être surchargé article par article depuis la colonne de réglages de l'éditeur Gutenberg.

## Fil d'Ariane

Le fil d'Ariane (breadcrumb) est un élément de navigation textuel qui indique à l'utilisateur sa position dans l'arborescence du site. Il s'affiche généralement en haut du contenu, sous l'en-tête.

<Frame>
  ![III 8 FIL D'ARIANE EXEMPLE](/images/III.8.FILD'ARIANEEXEMPLE.jpeg)
</Frame>

Il remplit deux fonctions complémentaires :

- Pour l'utilisateur : faciliter la navigation et le retour vers les sections parentes.
- Pour le référencement : les moteurs de recherche utilisent le fil d'Ariane pour mieux comprendre la structure du site. Il peut également s'afficher dans les résultats de recherche Google.

### **Activer le fil d'Ariane avec Astra**

Astra intègre nativement une fonctionnalité de fil d'Ariane, activable depuis Personnaliser \> Fil d'Ariane :

1. Activer l'option Activer le fil d'Ariane.
2. Choisir la position d'affichage : avant le titre, après le titre, ou désactivé sur certains types de pages.
3. Personnaliser le séparateur entre les niveaux (/, ›, →, etc.).
4. Cliquer sur Publier.
