Dans le cadre de l’épreuve E5B, plusieurs fonctionnalités attendues passent par les blocs Spectra (formulaire, diaporama, Google Maps), qui se comportent comme des widgets au sein de la page. Ces blocs se trouvent dans le sélecteur de blocs de Gutenberg (bouton +) sous la catégorie Spectra.

Pour chaque bloc Spectra :

Les options générales se trouvent dans le panneau latéral droit, onglet Bloc.

On y retrouve généralement des sections du type Général, Style, Avancé (marges, bordures, etc.).

## Formulaire de contact (Spectra Form)

Le bloc Form de Spectra permet de créer un formulaire de contact simple sans extension supplémentaire.

### Insertion du bloc

1. Dans l’éditeur, cliquer sur +.
2. Rechercher Form (Spectra).
3. Cliquer pour insérer le bloc dans la page (souvent sur la page Contact).

### Principales options (panneau Bloc à droite)

<u>Champs (onglet Général)</u> :

- Ajouter/supprimer des champs (nom, e-mail, message, téléphone, etc.).
- Choisir le type de champ (texte, email, zone de texte).
- Marquer un champ comme obligatoire.

<u>Bouton d’envoi</u> :

- Texte du bouton (ex. « Envoyer »).
- Alignement (gauche, centre, droite).

<u>Messages</u> :

- Texte de confirmation après envoi.
- Message d’erreur en cas de champ non rempli.

### L’onglet Style permet d’ajuster :

- Couleurs de fond et de texte.
- Bordures et coins arrondis.
- Espacing entre les champs.

## Galerie d’images

Pour afficher plusieurs images en grille, on peut utiliser :

Le bloc Galerie natif de WordPress.\
Ou un bloc Image Gallery de Spectra

### Insertion du bloc

1. Cliquer sur +.
2. Rechercher Galerie (ou Image Gallery).
3. Sélectionner les images dans la médiathèque ou téléverser de nouvelles images.

### Principales options (panneau Bloc)

- Nombre de colonnes.
- Espacement entre les images.
- Option Recadrage (pour uniformiser la hauteur/largeur).
- Affichage d’une légende sous chaque image.

Ces options permettent d’obtenir une présentation propre et lisible, utile pour illustrer un produit, un événement ou une équipe.

## Diaporama

Le bloc Slider de Spectra permet de créer un diaporama d’images ou de contenus.

### Insertion du bloc

1. Cliquer sur +.
2. Rechercher Slider (Spectra).
3. Insérer le bloc, puis ajouter des slides (diapositives) dans le bloc.

**Chaque slide peut contenir :**

- Une image d’arrière-plan.
- Un titre, un texte, un bouton (appel à l’action).

### Principales options (panneau Bloc)

<u>Général / Slider</u> :

- Nombre de slides visibles.
- Défilement automatique (autoplay) ou manuel.
- Vitesse de transition.

### Navigation

- Présence des flèches et des puces (points).

### Style

- Couleur et taille du texte.
- Alignement du contenu.
- Fond du slider (image, couleur).

## Google Maps

Le bloc Google Maps de Spectra permet d’afficher une carte interactive localisant l’entreprise.

### Insertion du bloc

1. Cliquer sur +.
2. Rechercher Google Maps (Spectra).
3. Insérer le bloc dans la page (par exemple, sur la page Contact).
